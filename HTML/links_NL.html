HELLO WORLD!

Hier zijn enkele belangrijke links die mijn programmeer cariere gelanceerd/verder geholpen hebben!

CodeCademy: Dit is de site waarop ik mijn eerste lijn code ooit geschreven heb. We moeten allemaal ergens starten en ik ben gestart bij HTML en CSS. Ook mijn allereerste lijn die ik op een site gekregen had was de traditionele zin "HELLO WORLD!" Deze zin moest gewoon mij startpunt zijn! Ondertussen heb ik al vele lessen afgewerkt en ik betrap er mezelf op om geregeld nog eens naar de site terug te keren om te herhalen of nieuwe dingen bij te leren. Heel erg bedankt CodeCademy, je was mijn perfecte start!

Udemy: Een van die "leer programmeren door het kijken van mensen die programmeren en zorg dat je het verstaat. Best leuk om je mee bezig te houden maar in tegenstelling tot CodeCademy heeft Udemy mij niet echt verder geholpen. Ze bieden gewoon niet dezelfde mogelijkheden als CodeCademy. Je krijgt geen oefeningen en ze motiveren je niet om buiten de video's zelf nog te proberen. Toch probeer ik via hun platform Python op mijn eentje te leren.

SkillValue: Op deze site ben ik eigenlijk een beetje per ongeluk gekomen. Ik had net de film The Enigma Code met Benedict Cumberbatch gezien, toen ik op Facebook een link zag: "Break enigma now!" Zo curieus als ik was, ben ik dan ook maar meteen de site gaan bezoeken. Een zeer interessante site kwam op mijn scherm. Er waren meer dan 100 uitdagingen voor zowel professionele programmeurs als voor studenten om hun coding skills te tonen. Als je de competitie zou winnen, kreeg je mooie prijzen opgestuurd. Ik had geprobeerd om enigma te kraken maar spijtig genoeg was ik nog niet goed genoeg. Misschien bij een volgende uitdaging dat ik kan meedoen

DataCamp: Deze site behandeld vooral data analyse en aangezien je voor data analyse Python moet kunnen (Python is de meest gebruikte taal voor data analyse en neural networking) kan ik op dit moment nog niet veel op de site doen. Ik ga er zeker weten naar terug keren van zodra ik wat meer Python ken!

Een laatste link die ik toch graag even in de schijnwerper zet is het YouTube channel van TheNetNinja. Deze man is heel erg creatief in het maken van websites en legt alles ook geweldig goed uit. Ook fijn om te horen is zijn typisch Brits accent! Hij heeft mij al een pak verder geholpen in het maken van mijn eigen website (deze dus) en ik zal zeker blijven kijken naar zijn tutorials (tot als ik het net zo goed kan als hem natuurlijk!).
