HELLO WORLD!

Hier zijn enkele belangrijke links die mij in de wereld van programmeren hebben gegooid!

CodeCademy: De site waar ik mijn eerste lijntjes code ooit heb geschreven! We moeten allemaal ergens starten en ik ben begonnen bij HTML. Ook voor mij was de eerste zin die ik op een website liet verschijnen niets anders dan (cf. title) HELLO WORLD! Ondertussen heb ik al heel wat cursussen achter de rug en ik zie er mezelf geregld nog eens oefeningen gaan maken! Dank je wel CodeCademy.

Udemy: Nog een "leer programmeren aan de hand van video's en kleine oefeningen". Zeer leuk om je bezig te houden wanneer je niet weet wat te doen! In tegenstelling tot CodeCademy weet ik niet of deze site veel bijbrengt. Okee je leert er wel vanalles op maar dat is door gewoon te kijken naar video's. Ze sporen je nooit aan om buiten deze video's te oefenen (in tegenstelling tot CodeCademy!). Desalniettemin probeer ik er toch Python mee te leren.

SkillValue: Op deze site ben ik eigenlijk een beetje perongeluk gekomen. Ik had net de film The Enigma Code met Benedict Cumberbatch gezien, toen ik op Facebook een link zag: "Break enigma now!" Zo curieus als ik was, ben ik dan ook maar meteen de site gaan bezoeken. Het komt er dus op need dat voor zowel student als professioneel programmeur, je hier mee kan doen aan soort van competities waarmee je prijzen kan winnen en eigenlijk op een leuke manier, programmeerprobelemen oplost. Spijtig genoeg ben ik nog niet goed genoeg om enigma te breken.

DataCamp: Hierover volgt later nog meer info aangezien ik nog geen tijd gehad heb om uit te zoeken wat de plus- en minpunten zijn van deze site. Op het eerste zicht ziet het er wel een mooi afgewerkte site uit die je verschillende mogelijkheden bied om Python the leren.

Een laatste link die ik toch graag even in de schijwerper zet is het YouTube channel van TheNetNinja. Deze man is heel erg creatief in het maken van websites en legt alles ook geweldig goed uit. Ook fijn om te horen is zijn typisch Brits accent! Hij heeft mij al een pak verder geholpen in het maken van mijn eigen website (deze dus) en ik zal zeker blijven kijken naar zijn tutorials (tot als ik het net zo goed kan als hem natuurlijk!).
